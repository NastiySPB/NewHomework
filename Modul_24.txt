# Оформление и оплата заказа
Цель: Получение заказа по указному адресу
## Основной сценарий:
1.	Пользователь открывает страницу с корзиной
2.	Система отображает страницу с корзиной, где показан список блюд и их топпингов.
3.	Пользовать подтверждает список выбранных блюд в корзине, открыв страницу для оформления заказа.
4.	Система отображает страницу с формой для заполнения необходимых данных - контактных данных, времени и адресе доставки
5.	Пользователь заполняет форму и запрашивает вариант оплаты заказа по карте
6.	Система отображает форму для ввода данных банковской карты
7.	Пользователь вводит данные банковской карты
8.	Система отображает страницу банковской операции для подтверждения платежа
9.	Пользователь подтверждает платеж
10.	Система отображает страницу с информацией о доставке оформленного заказа
## Дополнительные сценарии:
### Изменение топпингов у выбранных блюд
3. Пользователь запрашивает изменение топпинга к выбранному блюду из корзины
4. Система отображает страницу для выбора топпинга к выбранному блюду
5. Пользователь выбирает топпинг к блюду и сохраняет изменения
6. Возврат на шаг 2 основного сценария.
### Контактные данные и адрес подтягиваются автоматически
4. Система отображает страницу с уже заполненной формой с веденным контактными и адресными данными
5. Пользователь заполняет дополнительные поля и запрашивает вариант оплаты заказа по карте
6. Возврат на шаг 6 основанного сценария
### Оплата наличными:
5. Пользователь заполняет форму и вбирает вариант оплаты наличными
6. Система отображает форму для вводы суммы, с которой будет необходима сдача.
7. Пользователь заполняет форму и подтверждает вид оплаты
8. Система отображает страницу с информацией о доставке оформленного заказа
## Исключения:
### Пользователь не заполнил форму с адресом
6. Система отображает ошибку о том, что нужно заполнить все обязательные поля
### Пользователь не заполнил форму с информацией о сдаче в случае дополнительного сценария оплаты наличными
8. Система отображает ошибку о том, что нужно заполнить все обязательные поля
6. Система отображает ошибку
### Пользователь не заполнил форму с информацией о сдаче в случае дополнительного сценария оплаты наличными
8. Система отображает ошибку
### У пользователя не денег на счету
10. Система отображает ошибку.
